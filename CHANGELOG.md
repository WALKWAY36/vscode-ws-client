
# Change Log

All notable changes to the "WebSocket Client" extension will be documented in this file.

## [1.0.0] - 2026-02-21

### Added
- Initial release
- WebSocket connection support (ws:// and wss://)
- Message sending and receiving
- URL history with quick access
- Real-time message display with timestamps
- Connection status indicator
- Keyboard shortcuts (Ctrl+Shift+W to open)
- Dark/light theme support
- Auto-reconnect capability
- Message clearing functionality
